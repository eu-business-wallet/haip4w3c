# High Assurance Interoperability Profile for W3C VCDM 2.0 

## Abstract 
The goal of this joint community effort is to specify a profile of [W3C VCDM 2.0](https://www.w3.org/TR/vc-data-model-2.0/), 
which provides both a high level of assurance and interoperability, which can be contributed to suitable standardisation bodies such as 
[ETSI ESI](https://www.etsi.org/technical-groups/esi/) or [W3C VCWG](https://www.w3.org/groups/wg/vc/) for example.
